We are here for you, and always keen to help!

## Report a bug :bug: {#report-a-bug}

If you encountered a bug or something that feels like a bug, do not hesitate to [open a new bug](https://github.com/Sonia-corporation/stale/issues/new?assignees=&labels=bug+%3Abug%3A&template=bug_report.md&title=%5BBUG%5D+).

## Request a new feature :rocket: {#request-a-new-feature}

If you need an improvement or a completely new feature, do not hesitate to [open a new feature request](https://github.com/Sonia-corporation/stale/issues/new?assignees=&labels=feature-request+%3Arocket%3A&template=feature_request.md&title=%5BFEATURE%5D+).

## Improve the documentation :books: {#improve-the-documentation}

If you didn't understand an input or see a clear improvement regarding the documentation, you can [open a new documentation request](https://github.com/Sonia-corporation/stale/issues/new?assignees=&labels=docs-request+%3Abooks%3A&template=docs_request.md&title=%5BDOCS%5D+) to suggest how we can make this action easier to use, and understand.

## Improve the blog :books: {#improve-the-blog}

If for whatever reason you would like to contribute to the blog, you can [open a new blog post request](https://github.com/Sonia-corporation/stale/issues/new?assignees=C0ZEN&labels=blog-request+%3Anewspaper%3A&template=blog_article.md&title=%5BBLOG%5D+).

## Ask a question :question: {#ask-a-question}

If you simply need some help, [you can always ask us a question](https://github.com/Sonia-corporation/stale/discussions/new?category=q-a).

## Anything else :hole: {#anything-else}

If the previous sections were not enough to help you get sorted out, [you can always start a new discussion](https://github.com/Sonia-corporation/stale/discussions/new).
