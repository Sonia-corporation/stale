### Output {#output}

Name: `ignored-pull-requests-count`  
Type: `number`

### Description {#description}

This output will expose the number of pull requests ignored.

### Example {#example}

```yml {6}
# ...
steps:
  - name: Stale
    id: stale
    uses: sonia-corporation/stale@latest
  - name: Count
    run: echo "${{ steps.Stale.outputs.ignored-pull-requests-count }}"
```
