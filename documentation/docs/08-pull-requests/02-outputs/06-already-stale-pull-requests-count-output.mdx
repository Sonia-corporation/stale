### Output {#output}

Name: `already-stale-pull-requests-count`  
Type: `number`

### Description {#description}

This output will expose the number of pull requests processed which were already stale.

### Example {#example}

```yml {6}
# ...
steps:
  - name: Stale
    id: stale
    uses: sonia-corporation/stale@latest
  - name: Count
    run: echo "${{ steps.Stale.outputs.already-stale-pull-requests-count }}"
```
