### Output {#output}

Name: `called-api-pull-requests-mutations-count`  
Type: `number`

### Description {#description}

This output will expose the number of GitHub API mutations calls performed for the pull requests.

### Example {#example}

```yml {6-7}
# ...
steps:
  - name: Stale
    id: stale
    uses: sonia-corporation/stale@latest
  - name: Count
    run: echo "${{ steps.Stale.outputs.called-api-pull-requests-mutations-count }}"
```
