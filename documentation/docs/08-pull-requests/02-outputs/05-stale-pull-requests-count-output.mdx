### Output {#output}

Name: `stale-pull-requests-count`  
Type: `number`

### Description {#description}

This output will expose the number of pull requests stale.

### Example {#example}

```yml {6-7}
# ...
steps:
  - name: Stale
    id: stale
    uses: sonia-corporation/stale@latest
  - name: Count
    run: echo "${{ steps.Stale.outputs.stale-pull-requests-count }}"
```
