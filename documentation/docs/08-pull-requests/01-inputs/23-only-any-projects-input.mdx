### Input {#input}

Name: `pull-request-only-any-projects`  
Type: `string[]`  
Default value: `[]`

### Description {#description}

This input will let you process only the pull requests that are linked to a project which is white-listed with this input.  
This can be useful when you use the projects for triage.

### Example {#example}

```yml {6-8}
# ...
name: Stale
id: stale
uses: sonia-corporation/stale@latest
with:
  pull-request-only-any-projects: |
    project-x
    project-y
```
