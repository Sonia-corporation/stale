### Input {#input}

Name: `issue-only-with-projects`  
Type: `boolean`  
Default value: `false`

### Description {#description}

This input will let you process only the issues that are linked to at least one project.  
This can be useful when you use the projects for triage.

### Example {#example}

```yml {6}
# ...
name: Stale
id: stale
uses: sonia-corporation/stale@latest
with:
  issue-only-with-projects: true
```
