### Input {#input}

Name: `issue-close-comment`  
Type: `string`  
Default value: `''`

### Description {#description}

This input will let you define the comment that will be added on the issues when they are closed after being stale for too long during the processing.  
This can be useful if you wish to improve the communication and the process on how to handle an unwanted closed issue.

:::info

You can also specify the reason for closing this issue by using the [close reason input](issue-close-reason-input),
add labels onto it by using the [add labels after close input](issue-add-labels-after-close-input),
remove labels onto it by using the [remove labels after close input](issue-remove-labels-after-close-input)
and define the number of days before closing it by using the [days before close input](issue-days-before-close-input).

:::

### Example {#example}

```yml {6-7}
# ...
name: Stale
id: stale
uses: sonia-corporation/stale@latest
with:
  issue-close-comment: |
    This issue is now closed due to a lack of activity!
```
