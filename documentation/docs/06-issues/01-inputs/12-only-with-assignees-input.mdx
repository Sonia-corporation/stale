### Input {#input}

Name: `issue-only-with-assignees`  
Type: `boolean`  
Default value: `false`

### Description {#description}

This input will let you process only the issues that are linked to at least one assignee.  
This can be useful when you use the assignees for triage.

### Example {#example}

```yml {6}
# ...
name: Stale
id: stale
uses: sonia-corporation/stale@latest
with:
  issue-only-with-assignees: true
```
