### Input {#input}

Name: `issue-only-any-milestones`  
Type: `string[]`  
Default value: `[]`

### Description {#description}

This input will let you process only the issues that are linked to a milestone which is white-listed with this input.  
This can be useful when you use the milestones for triage.

### Example {#example}

```yml {6-8}
# ...
name: Stale
id: stale
uses: sonia-corporation/stale@latest
with:
  issue-only-any-milestones: |
    milestone-x
    milestone-y
```
