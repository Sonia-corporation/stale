### Input {#input}

Name: `issue-ignore-any-labels`  
Type: `string[]`  
Default value: `[]`

### Description {#description}

This option will let you ignore the processing of the issues which have at least one of the label from this list.  
This can be useful when you use labels for triage.

### Example {#example}

```yml {6-8}
# ...
name: Stale
id: stale
uses: sonia-corporation/stale@latest
with:
  issue-ignore-any-labels: |
    need help
    bug
```
