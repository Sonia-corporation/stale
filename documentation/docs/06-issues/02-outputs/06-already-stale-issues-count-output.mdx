### Output {#output}

Name: `already-stale-issues-count`  
Type: `number`

### Description {#description}

This output will expose the number of issues processed which were already stale.

### Example {#example}

```yml {6-7}
# ...
steps:
  - name: Stale
    id: stale
    uses: sonia-corporation/stale@latest
  - name: Count
    run: echo "${{ steps.Stale.outputs.already-stale-issues-count }}"
```
