### Output {#output}

Name: `issues-labels-count`  
Type: `number`

### Description {#description}

This output will expose the number of mutated issues labels (added or removed).

### Example {#example}

```yml {6-7}
# ...
steps:
  - name: Stale
    id: stale
    uses: sonia-corporation/stale@latest
  - name: Count
    run: echo "${{ steps.Stale.outputs.issues-labels-count }}"
```
