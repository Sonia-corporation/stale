### Output {#output}

Name: `unaltered-issues-count`  
Type: `number`

### Description {#description}

This output will expose the number of issues unaltered (either not good to stale or already stale).

### Example {#example}

```yml {6-7}
# ...
steps:
  - name: Stale
    id: stale
    uses: sonia-corporation/stale@latest
  - name: Count
    run: echo "${{ steps.Stale.outputs.unaltered-issues-count }}"
```
