### Output {#output}

Name: `processed-issues-count`  
Type: `number`

### Description {#description}

This output will expose the number of issues processed.

### Example {#example}

```yml {6}
# ...
steps:
  - name: Stale
    id: stale
    uses: sonia-corporation/stale@latest
  - name: Count
    run: echo "${{ steps.Stale.outputs.processed-issues-count }}"
```
