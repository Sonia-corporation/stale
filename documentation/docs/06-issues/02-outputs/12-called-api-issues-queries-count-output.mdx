### Output {#output}

Name: `called-api-issues-queries-count`  
Type: `number`

### Description {#description}

This output will expose the number of GitHub API queries calls performed for the issues.

### Example {#example}

```yml {6}
# ...
steps:
  - name: Stale
    id: stale
    uses: sonia-corporation/stale@latest
  - name: Count
    run: echo "${{ steps.Stale.outputs.called-api-issues-queries-count }}"
```
