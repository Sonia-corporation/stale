### Output {#output}

Name: `deleted-pull-requests-branches-count`  
Type: `number`

### Description {#description}

This output will expose the number of deleted pull requests branches closed.

### Example {#example}

```yml {6}
# ...
steps:
  - name: Stale
    id: stale
    uses: sonia-corporation/stale@latest
  - name: Count
    run: echo "${{ steps.Stale.outputs.deleted-pull-requests-branches-count }}"
```
