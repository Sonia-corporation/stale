### Output {#output}

Name: `draft-pull-requests-count`  
Type: `number`

### Description {#description}

This output will expose the number of added pull requests labels.

### Example {#example}

```yml {6}
# ...
steps:
  - name: Stale
    id: stale
    uses: sonia-corporation/stale@latest
  - name: Count
    run: echo "${{ steps.Stale.outputs.draft-pull-requests-count }}"
```
