### Input {#input}

Name: `pull-request-ignore-all-labels`  
Type: `boolean`  
Default value: `false`

### Description {#description}

This option will let you ignore the processing of the pull requests which have at least one label.  
This can be useful when you use labels for triage.

:::note

If the pull request is stale and so, has a [stale label](pull-request-stale-label-option), this option will not ignore the processing.

:::

### Example {#example}

```yml {6}
# ...
name: Stale
id: stale
uses: sonia-corporation/stale@latest
with:
  pull-request-ignore-all-labels: true
```
