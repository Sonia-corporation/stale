### Input {#input}

Name: `pull-request-days-before-close`  
Type: `number`  
Default value: `30`

### Description {#description}

This option will let you define the number of days before processing the pull requests to close.

The number of days is calculated based on the stale date (the stale label creation date) from today.

**For example:**

Assuming that a pull request is stale since 9 days and this option is configured to close after 10 days.  
The pull request will be closed tomorrow if no update occur until then.

### Example {#example}

```yml {6}
# ...
name: Stale
id: stale
uses: sonia-corporation/stale@latest
with:
  pull-request-days-before-close: 10
```
