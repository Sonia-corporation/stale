The list of all the pull requests options.

| Input                                                                                       | Description                                                                                                                                            | Default |
| ------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------ | ------- |
| [pull-request-stale-label](pull-request-stale-label-option)                                 | The label that will be added to the pull request when it is stale.                                                                                     | `stale` |
| [pull-request-ignore-all-labels](pull-request-ignore-all-labels-option)                     | Allow to ignore the processing of pull requests that contains any labels.                                                                              | `false` |
| [pull-request-ignore-any-labels](pull-request-ignore-any-labels-option)                     | Allow to ignore the processing of pull requests that contains one of those labels (multiline).                                                         |         |
| [pull-request-ignore-all-assignees](pull-request-ignore-all-assignees-option)               | Allow to ignore the processing of pull requests that contains any assignees.                                                                           | `false` |
| [pull-request-ignore-any-assignees](pull-request-ignore-any-assignees-option)               | Allow to ignore the processing of pull requests that contains one of those assignees (multiline).                                                      |         |
| [pull-request-ignore-all-project-cards](pull-request-ignore-all-project-cards-option)       | Allow to ignore the processing of pull requests that contains any project cards.                                                                       | `false` |
| [pull-request-ignore-before-creation-date](pull-request-ignore-before-creation-date-option) | Allow to ignore the processing of pull requests that were created before this date ([ISO 8601](https://moment.github.io/luxon/#/parsing?id=iso-8601)). |         |
| [pull-request-ignore-draft](pull-request-ignore-draft-option)                               | Allow to ignore the processing of pull requests that are drafts.                                                                                       | `false` |
| [pull-request-days-before-stale](pull-request-days-before-stale-option)                     | The number of days until the pull request is considered as stale.                                                                                      | `30`    |
| [pull-request-stale-comment](pull-request-stale-comment-option)                             | The comment that will be sent once the pull request is stale (keep empty to not send a comment).                                                       |         |
| [pull-request-days-before-close](pull-request-days-before-close-option)                     | The number of days until a stale pull request is considered as closed.                                                                                 | `10`    |
| [pull-request-close-comment](pull-request-close-comment-option)                             | The comment that will be sent once the pull request is close (keep empty to not send a comment).                                                       |         |
| [pull-request-delete-branch-after-close](pull-request-delete-branch-after-close-option)     | Delete the branch when the processing close the pull request.                                                                                          | `false` |
| [pull-request-add-labels-after-stale](pull-request-add-labels-after-stale-option)           | A list of labels added when the processing stale the pull request (multiline).                                                                         |         |
| [pull-request-add-labels-after-close](pull-request-add-labels-after-close-option)           | A list of labels added when the processing close the pull request (multiline).                                                                         |         |
| [pull-request-processing](pull-request-processing-option)                                   | Allow to process the pull requests.                                                                                                                    | `true`  |
| [pull-request-to-draft-instead-of-stale](pull-request-to-draft-instead-of-stale-option)     | Convert the pull request to a draft pull request instead of handling it as a stale candidate.                                                          | `false` |
